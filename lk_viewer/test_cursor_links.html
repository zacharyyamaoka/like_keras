How to Use:

  1. Click any of the links above
  2. Your browser will ask if you want to open Cursor
  3. Click "Allow" or "Open Cursor"
  4. The file should open in Cursor

URL Format:

The Cursor URL handler follows this format:

cursor://file/<absolute-path>:<line>

Examples:

  • cursor://file//home/user/project/file.py - Opens file.py
  • cursor://file//home/user/project/file.py:42 - Opens file.py at line 42
Troubleshooting: If the links don't work:
  • Make sure Cursor is installed on your system
  • Try restarting Cursor and your browser
  • On Linux, you may need to register the URL handler manually
