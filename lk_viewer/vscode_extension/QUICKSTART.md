## Quick Start Guide

### 1. Open Extension in Cursor

```bash
cursor /home/bam/bam_ws/src/like_keras/lk_viewer/vscode_extension
```

### 2. Launch Extension (Two Ways)

**Method A: Press F5**
- Opens Extension Development Host
- Extension auto-loads

**Method B: Via Command**
- Press `Cmd/Ctrl + Shift + P`
- Type "Debug: Start Debugging"
- Select "Run Extension"

### 3. Test It!

In the new window:

**Sidebar Method:**
- Look for the file icon in the left sidebar (Activity Bar)
- Click to open "LK Viewer: File Navigator"
- Click any file link - **INSTANT opening!**

**Command Palette Method:**
- Press `Cmd/Ctrl + Shift + P`  
- Type: "LK Viewer: Open File Panel"
- Opens in editor area

**Test Command:**
- Press `Cmd/Ctrl + Shift + P`
- Type: "LK Viewer: Open File (for testing)"
- Opens features.md:10 instantly!

### 4. See the Speed!

The timing display shows:
- Message send time
- File open time  
- Total: **~5-10ms** (100x faster than browser!)
